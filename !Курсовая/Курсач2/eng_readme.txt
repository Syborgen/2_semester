Operator's manual
Program purposes:
The program is designed to simulate Air traffic.
The conditions of program execution:
To run the program requires a PC type IBM PC or software compatible with it, running the operating system of the Windows family (not lower than Windows XP).
Program execution:
To switch the language, the user must press the key specified in the menu.
Select the desired menu item by pressing the number keys (0-9). Use the ESC key to return to the previous menu. If you press 9 in the main menu, the program will exit.
