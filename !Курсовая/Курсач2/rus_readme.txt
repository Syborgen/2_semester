Руководство оператора
Назначение программы:
Программа предназначена для моделирования Воздушного трафика.
Условия выполнения программы:
Для выполнения программы необходима ПЭВМ типа IBM PC или программно совместимая с ней, функционирующая под управлением операционной системы семейства Windows (не ниже Windows XP).
Выполнение программы:
Для переключения языка пользователь должен нажать указаную в меню клавишу.
Выбор необходимого пункта меню осуществляется путем нажатия клавиш с числами (0-8). Для возврата в предыдущее меню используется клавиша ESC. При нажатии 9 в главном меню программа завершит свою работу.
