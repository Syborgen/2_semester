Руководство оператора 
Назначение программы:
Программа предназначена для моделирования процесса взаимодействия пользователя с банковскими аккаунтами.
Условия выполнения программы:
Для выполнения программы необходима ПЭВМ типа IBM PC или программно совместимая с ней, функционирующая под управлением операционной системы семейства Windows (не ниже Windows XP). 
Выполнение программы:
Для переключения языка пользователь должен нажать нужно нажать указаную в меню клавишу.
Выбор необходимого пункта меню осуществляется путем нажатия клавиш с числами (1-9). Для возврата в предыдущее меню используется клавиша 0. При нажатии 0 в главном меню программа завершит свою работу.
